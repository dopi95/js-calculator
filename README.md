# js-calculator

A clean and modern calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations: `+`, `-`, `×`, `÷`
- Percentage support
- Delete last character (⌫) and clear all (AC)
- Live expression display
- Keyboard support
- Glassmorphism UI design

## Tech Stack
- HTML5
- CSS3 (Glassmorphism, Grid, Transitions)
- Vanilla JavaScript

## How to Use
Open `index.html` in any browser — no setup needed.
